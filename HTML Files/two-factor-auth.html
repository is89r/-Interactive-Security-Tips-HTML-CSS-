Enable Two-Factor Authentication

Adding an extra layer of security to your accounts

What is Two-Factor Authentication (2FA)?

2FA is a security process in which the user provides two different authentication factors to verify themselves. It adds an extra step in logging in, making it harder for attackers to access your account.

Why is 2FA important?

Even if your password is stolen, 2FA helps protect your account by requiring a second step—like a code from an app or SMS. It reduces the risk of unauthorized access.

Common types of 2FA:

  • Authentication apps (e.g., Google Authenticator, Authy)
  • SMS verification codes
  • Hardware security keys (e.g., YubiKey)

How to get started:

Check your account security settings and enable 2FA wherever it’s supported. Use an app for better security instead of SMS if possible.
